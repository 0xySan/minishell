# minishell
My minishell ( 21SH ) for the 42 Cursus

## Features
- Command execution with arguments.
- Basic built-in commands (`cd`, `echo`, `exit`, etc.).
- Environment variable handling.
- Piping and redirection support.
- Signal handling (e.g., Ctrl+C, Ctrl+D).
